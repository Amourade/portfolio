# Personal Portfolio

## How to setup

1. Change .env.example to .env and adjust database information
2. `composer install`
3. `php artisan key:generate`
4. `php artisan migrate`
5. `php artisan db:seed`
6. `php artisan storage:link`
7. `npm run install`
8. `php artisan serve`
9. `npm run dev`
